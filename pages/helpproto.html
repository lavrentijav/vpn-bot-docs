Документация по VPN-протоколам и подключению клиентов

Здесь представлены инструкции по настройке популярных VPN-протоколов и подключению через рекомендованные клиентские приложения.

VLESS

Описание

Современный и быстрый протокол Xray, обеспечивающий высокую скорость и надежность.

Рекомендуемые клиенты и инструкция подключения

Android — Nekobox

  1. Скачайте и установите приложение с сайта github: Nekobox.
  2. Скопируйте ссылку из бота
  3. Запустите Nekobox и нажмите «Добавить профиль»(значок файла с плюсом)
  4. Выбирите «Импортировать из буфера обмена»
  5. Готово. Можно подключиться

iOS / macOS — Foxray

  1. Скачайте Foxray из App Store: Foxray.
  2. Откройте приложение и нажмите на "+" для добавления новой конфигурации.
  3. Выберите тип подключения VLESS.
  4. Введите данные сервера (адрес, порт, UUID и другие параметры).
  5. Сохраните и подключитесь.

Windows — Invisible Man Xray

  1. Скачайте последнюю версию с GitHub: Invisible Man Xray.
  2. Распакуйте архив и запустите программу.
  3. Нажмите «Управление конфигурациями» → → → .
  4. нажмите +.
  5. выберите Импортировать по ссылке
  6. Вставте ссылку полученную от бота
  7. Готово. Можно подключиться

Shadowsocks

Описание

Легкий SOCKS5-прокси с шифрованием. Простая настройка, поддерживается множеством клиентов.

Рекомендуемые клиенты и инструкция подключения

Android — Nekobox

Шаги аналогичны подключению VLESS, просто выберите тип протокола Shadowsocks.

iOS / macOS — Shadowrocket

  1. Скачайте приложение Shadowrocket из App Store.
  2. Откройте приложение и добавьте новый сервер.
  3. Выберите тип Shadowsocks.
  4. Введите параметры (сервер, порт, пароль, метод шифрования).
  5. Сохраните и подключитесь.

Windows — Shadowsocks‑Win

  1. Скачайте клиент с GitHub: Shadowsocks‑Win.
  2. Запустите программу.
  3. В настройках добавьте новый профиль Shadowsocks с параметрами сервера.
  4. Сохраните и включите прокси.

Outline

Описание

Полное решение от Jigsaw с собственным сервером и приложениями, простое в использовании.

Инструкция

  1. Перейдите на сайт: https://getoutline.org/ru/home.
  2. Скачайте и установите Outline Client для вашей платформы.
  3. Получите доступ к серверу через Outline Manager или ссылку от администратора.
  4. Импортируйте ключ сервера в клиент.
  5. Подключитесь и пользуйтесь VPN.

HTTP-прокси

Описание

Простой нешифрованный HTTP-прокси для базовых задач.

Клиенты и инструкция

Android — ProxyDroid

  1. Установите ProxyDroid из Google Play: ProxyDroid.
  2. Запустите приложение.
  3. Введите адрес и порт HTTP-прокси.
  4. Включите прокси и настройте фильтрацию приложений, если нужно.
